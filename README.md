# FotballQuiz – MVP

En enkel fotballquiz‑webapp bygget med React + Vite + TypeScript + Tailwind + Framer Motion.

## Kommandoer

```bash
npm install
npm run dev
# build
npm run build
npm run preview
```

## Hva er inkludert?
- Velg favorittklubb (knapper + eget klubbnavn)
- Historie‑fane (demo‑tekst)
- Quiz‑fane (flervalg med forklaring, score, retry)
- Enkle UI‑komponenter (Button, Card, Tabs, Input, RadioGroup) – ingen shadcn‑install nødvendig

## Videre arbeid
- Backend (Node/Express + MongoDB) for spørsmål, brukere, highscores
- Admin‑UI for å legge inn spørsmål og historietekster
- Autogenerere spørsmål fra kilder (Wikipedia/DBpedia/Offisielle sider)
- Daglige utfordringer, tidsmodus, leaderboards per klubb
